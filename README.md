# HabitCLI

A minimalist habit tracking web application with a command-line inspired interface. Track your daily habits, build streaks, and stay consistent with a developer-friendly UI.

## Project Vision

HabitCLI helps users build and maintain habits through a clean, terminal-style interface. Users can add habits, mark them complete each day, and visualize their progress with streak tracking and simple analytics.

## Target Users

- Developers and tech enthusiasts who prefer keyboard-driven interfaces
- Anyone looking for a distraction-free habit tracker
- Users who want quick habit logging without complex features

## Tech Stack

- Frontend: Next.js 16, React 19, TypeScript
- Styling: Tailwind CSS
- State Management: React hooks
- Data Storage: localStorage (client-side)

## Features (Planned)

- Add/edit/delete habits
- Daily habit check-ins
- Streak tracking
- Simple analytics dashboard
- CLI-inspired dark theme interface
- Keyboard shortcuts for power users

## Getting Started

Install dependencies:
npm install

Run the development server:
npm run dev

Open http://localhost:3000 in your browser.
